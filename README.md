## The Problem
When someone gets sick or injured in a foreign country, they face three critical failures:
- Cannot explain symptoms in the local language
- Don't know which hospital to go to
- Cannot communicate during emergency calls

No existing app solves all three cleanly.

## Core Features
| Feature | Description |
|---|---|
| 🚨 One-Tap Emergency | Calls 119 (Korea) with pre-translated location-aware message |
| 🏥 Hospital Finder | Real-time nearest hospital using Google Maps + Places API |
| 📋 Medical Card Generator | AI-powered multilingual card with blood type, allergies, medications |

## Tech Stack
| Layer | Technology |
|---|---|
| Language | Kotlin |
| UI | Jetpack Compose |
| Database | Firebase Firestore + Room DB |
| AI / NLP | Gemini API |
| Maps | Google Maps SDK + Places API |
| Notifications | Firebase Cloud Messaging |

## Research Vision
Aegis is piloted for South Korea but designed as a globally scalable system. The underlying research explores adaptive mobile sensing for emergency contexts — how a mobile system can intelligently respond to real-world environmental and physiological context in life-critical situations.
